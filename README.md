⚽ Football Match Outcome Prediction and Team Performance Analysis 

📌 Project Overview

This project focuses on predicting football match outcomes using historical match data collected from four major football leagues across approximately twenty seasons per league. The dataset was built by aggregating and preprocessing more than 80 individual season files into a unified analytical database.

The objective was to predict whether the home team would win a match (1 = Home Win, 0 = Otherwise) using machine learning, deep learning, and unsupervised learning techniques.


🎯 Objectives 

Collect and integrate multi-season football match data from multiple leagues.
Build a large-scale dataset for predictive analytics.
Predict home team victories using supervised machine learning models.
Compare the performance of multiple classification algorithms.
Identify team similarities and performance patterns using clustering techniques.
Explore hidden structures within football data through dimensionality reduction. 


📊 Dataset 

Four major football leagues
Approximately 20 seasons per league
More than 80 source files combined into a single dataset
Historical match statistics and performance indicators
Binary target variable:
1 = Home Team Win
0 = Draw or Away Team Win 


🛠 Data Preparation 

Merged and standardized data from multiple seasons and leagues. 

Cleaned missing and inconsistent values. 

Performed feature engineering and preprocessing. 

Encoded categorical variables. 

Scaled and transformed features where necessary.
